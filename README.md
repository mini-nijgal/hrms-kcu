## Features

- Attendance System
- Employee Management
- Resign Management
- Leave Management
- Notice board 
- Multi Approval System
- Payroll

- Laravel 9 - PHP Framework for building the admin dashboard and mobile api.
- VueJS 3 - The Progressive JavaScript Framework integrated with laravel and inertia.
- Tailwind CSS.
